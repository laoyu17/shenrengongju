# I1 执行说明：口径冻结 + CI门禁

本文档对应 `review/04-两迭代收口路线图.md` 中 I1-1 与 I1-2 的落地实现。

## 1. 目录结构

- 冻结数据集与清单
  - `review/frozen/sched_rate/configs/**.yaml`（12个固定样本）
  - `review/frozen/sched_rate/config-list.txt`（唯一权威样本列表）
  - `review/frozen/sched_rate/manifest.tsv`（sha256与大小）
- 可复跑脚本
  - `review/scripts/i1_freeze_sched_rate_gate.sh`（I1-1）
  - `review/scripts/i1_ci_gate.sh`（I1-2）

## 2. I1-1：口径冻结脚本

命令：

```bash
bash review/scripts/i1_freeze_sched_rate_gate.sh
```

可选：

```bash
TARGET_UPLIFT=0.30 bash review/scripts/i1_freeze_sched_rate_gate.sh review/runtime/i1/sched_rate_gate
```

固定口径：
- 数据集来源：`review/frozen/sched_rate/config-list.txt`
- baseline：`np_edf`
- candidates：`np_dm,precautious_dm,lp`
- task_scope：`sync_only`
- 门禁指标：`candidate_only_uplift`

输出：
- `benchmark_frozen.json`
- `benchmark_frozen.csv`
- `summary.txt`（含 `gate_metric=candidate_only_uplift`）

通过标准：
- 报告包含 `candidate_only_uplift` 与 `candidate_only_schedulable_rate`
- `total_cases == config-list 行数`
- `candidate_only_uplift >= TARGET_UPLIFT`

## 3. I1-2：CI门禁脚本

命令：

```bash
bash review/scripts/i1_ci_gate.sh
```

可选输出目录：

```bash
bash review/scripts/i1_ci_gate.sh review/runtime/i1/ci_gate
```

串行步骤：
1. `python -m pytest -q`
2. `validate`（at01）
3. `run`（at01）
4. `inspect-model --strict-on-fail`（at01）
5. `inspect-model --strict-on-fail`（at01~at10）
6. `plan-static`（at06）
7. `analyze-wcrt --strict-plan-match`（at06）
8. `export-os-config --strict-plan-match`（at06）
9. 调用 I1-1 冻结口径脚本

输出：
- `review/runtime/i1/ci_gate/results.tsv`
- `review/runtime/i1/ci_gate/logs/*.log`
- `review/runtime/i1/ci_gate/artifacts/**`

返回码：
- 全部通过返回 `0`
- 任一步失败返回 `1`
